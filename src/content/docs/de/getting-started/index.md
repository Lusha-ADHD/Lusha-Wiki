Lusha ist ein mobiles Spiel von Dygie, das Kindern und Familien hilft, Alltagsroutinen motivierender zu gestalten. Es verbindet reale Aufgaben, etwa Selbstfürsorge, Aufgaben im Haushalt oder Momente zur Emotionsregulation, mit spielerischem Fortschritt in einer Abenteuerwelt.

## Für wen ist Lusha gedacht?

Lusha richtet sich vor allem an Familien mit Kindern. Ein Elternteil oder eine vertraute erwachsene Person kann die Routinen begleiten, während das Kind das Spiel erlebt, Quests abschließt und ermutigt wird.

## Vor dem Start

- Verwenden Sie eine offizielle Quelle, wenn Sie Lusha installieren oder Informationen dazu lesen.
- Bereiten Sie wenige einfache Routinen vor, zum Beispiel eine Morgenaufgabe, eine Abendaufgabe oder eine beruhigende Handlung.
- Stellen Sie Lusha als Unterstützung und Spiel vor, nicht als Strafe oder Test.
- Beginnen Sie realistisch: Eine kleine, verlässlich begleitete Routine ist besser als eine lange Liste.

## Empfohlene erste Sitzung

1. Erkunden Sie die App gemeinsam mit dem Kind.
2. Wählen Sie eine oder zwei konkrete Aufgaben aus.
3. Erklären Sie, wann eine Aufgabe bestätigt wird und was danach im Spiel passiert.
4. Besprechen Sie gemeinsam, was leicht war, was schwierig war und was zum Weitermachen motiviert.
