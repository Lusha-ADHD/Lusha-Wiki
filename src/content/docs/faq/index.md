## Starter Questions

### What is Lusha?

Lusha is a game project with a dedicated wiki for onboarding, world information, frequently asked questions, and clinical trial context.

### Is this the official wiki?

Yes. This site is intended to be the canonical public wiki for Lusha.

### Which languages are supported?

The initial language set is English, French, German, Italian, Spanish, and Brazilian Portuguese.

### Where should clinical trial questions go?

Clinical trial topics should be covered in the Clinical Trial Info section with clear dates, sources, and disclaimers.
